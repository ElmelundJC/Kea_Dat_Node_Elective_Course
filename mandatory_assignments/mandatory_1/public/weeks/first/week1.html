JavaScript Basics

Different Datatypes:

Primitives:

  • Undefined - variable has not been assigned a value
  • Boolean - true or false
  • Number - integer, float, etc.
  • Null - value null
  • String - set of characters
  • Symbol - a unique and immutable primitive value

Structural types:

  • Object - allows you to store collections of data
  • Function - the function is a callable object that executes a block of code. Possible to be assigned to a variable.
  • Array - storing multiple values in a single variable

In doubt what type you are working with... Just use the magic method typeof()

What is NodeJS?

Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser. How do to create a node project

  1. Create package.json file
  2. Define {} inside the package.json file, which tells that the file is of type JSON
  3. Run the command in terminal: "npm install" to create package-lock-.json
  4. Two possible ways:
    • Inside package.json you should define your dependencies as key:value pair, and as the value you should define express and the version you would like to install to your project. Run the command in terminal: "npm install" to install the project and its dependencies. Inside the directory folder you should see a map called node_modules which is the express files and librarys are.
    • Or you could run the command in terminal: "npm install express", which does exactly the same as explained above
  5. Create app.js file
  6. Inside app.js file, import the express library, & save the library in a variable & define what port the application should run on.
    "const express = require("express");"
    "const app = express();"
    "app.listen(8080);"

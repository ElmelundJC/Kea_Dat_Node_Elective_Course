For Loops we have been talking about maps, and gotten an example of how to loop through a map.

Example:

const fruits = ["apples", "banana", "blueberries"];

using maps to loop through the array by doing:

fruits.map(fruit => console.log(fruit));

Also looping throug the array while adding data to the elements:

const copiedFruits = fruits.map((fruit, index) => {
return { id: index + 1, fruit, delicious: "yes yes yes" };
});
console.log(copiedFruits);
